## Watermark Explanation

The watermark `PRIMAX-AI-ELOHIM-BSP-2025` appears in:
- HTML comments
- JavaScript headers
- Footer text
- JSON metadata

**What it means:**
- `PRIMAX-AI` — From the Primax AI system (elohim-forge project)
- `ELOHIM` — This game's name (Elohim Shards)
- `BSP` — Bakery Street Project (author organization)
- `2025` — Year of creation

This watermark serves as:
1. **Provenance marker** — Identifies the source/author
2. **Copyright notice** — Claims ownership
3. **Version identifier** — Tracks project generation
4. **Tamper detection** — If removed, indicates unauthorized modification
