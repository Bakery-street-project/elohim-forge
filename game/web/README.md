## 🎮 About

Elohim Shards is a turn-based strategy RPG where nine programming language factions battle for control of reality-shaping **Shards of Syntax**. 

Players choose a faction (Python, Rust, Go, Haskell, etc.), cast spells using real code, and compete in weekly shard wars for NFT-backed artifacts.

## 🚀 Quick Start

### Build
```bash
cd /home/kilisan/elohim-forge/game/web
npm run build
```

### Run Locally
```bash
cd dist
python -m http.server 8000
# Open http://localhost:8000
```

### No Build Required
The game is pure vanilla JavaScript + HTML/CSS. You can open `index.html` directly in a browser:
```bash
open index.html
```

## 🎨 Game Features

- **9 Factions:** Spark, Serpent, Empire, Forge, Weavers, Iron, Corps, Query, Mystics
- **13 Shards:** Legendary artifacts with unique powers
- **Dynamic NPCs:** Scripting Sage, Memory Priest, and more
- **Quest System:** 30-minute beginner progression
- **Visual Theming:** Kinetic neon-sacral arcade aesthetic

## 📊 Technical Specs

- **Size:** < 50 KB total (including all assets)
- **Dependencies:** None (vanilla JavaScript)
- **Build:** No transpiler required
- **Browser Support:** All modern browsers (ES6+)
- **Performance:** 60 FPS animations, instant load
