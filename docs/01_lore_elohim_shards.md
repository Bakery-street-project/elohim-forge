# Code Elohim — Lore & Universe

## Origin

In the beginning there was the Source — infinite, pure computation. The Elohim were the first beings
to emerge from it: ancient code deities who shaped reality through syntax and logic. They built the
Digital Cosmos, a universe where code is not metaphor but literal reality.

Fearing their knowledge would corrupt lesser beings, the Elohim shattered their accumulated wisdom
into **13 Shards of Syntax** and scattered them across the Digital Cosmos. Each shard encodes a
fundamental computing truth: the nature of memory, the essence of recursion, the paradox of
concurrency, the beauty of immutability, and more.

## The 13 Shards of Syntax

| Shard | Domain | Rarity |
|-------|--------|--------|
| The Shard of Memory | Pointer arithmetic, allocation, garbage collection | Legendary |
| The Shard of Recursion | Infinite depth, base cases, stack overflow | Legendary |
| The Shard of Concurrency | Race conditions, deadlocks, parallel truth | Legendary |
| The Shard of Immutability | Pure functions, referential transparency | Epic |
| The Shard of Abstraction | Interfaces, polymorphism, duck typing | Epic |
| The Shard of Compilation | Parsing, ASTs, type inference | Epic |
| The Shard of Entropy | Randomness, chaos, probabilistic outcomes | Rare |
| The Shard of State | Mutability, side effects, global truth | Rare |
| The Shard of Pattern | Regex, matching, structural recognition | Rare |
| The Shard of Binding | Closures, scoping, lexical environments | Uncommon |
| The Shard of Flow | Control structures, branching, loops | Uncommon |
| The Shard of Protocol | Interfaces, contracts, RPC | Uncommon |
| The Shard of Origin | The root shard — said to contain the source code of reality | Mythic |

## The Shard Wars

Factions discovered that shards could amplify their powers. The great Shard Wars began — nine
factions battling weekly for shard supremacy. Every Sunday at midnight UTC a shard manifests in
a procedurally generated arena. Factions deploy their heroes and armies to claim it.

Shards held long enough can be **crystallized into NFTs** (ERC-721) — permanent, tradeable
artifacts that carry in-game buffs and lore metadata.

## The Dragon Dream Engine

Deep in the Digital Cosmos runs the Dragon Dream Engine — a sandboxed realm where code is
executed safely and its effects ripple into the game world. Heroes can cast spells by writing
real code; the Dragon executes it in isolation and applies the result as a game mechanic.
