# The 13 Shards of Syntax

The Shards are more than artifacts; they are the fundamental constants of reality. Each Shard contains a portion of the Elohim's original design, capable of altering the laws of the Operational Plane when activated.

| Shard | Domain | Rarity | Effect |
|-------|--------|--------|--------|
| **Memory** | Allocation & Pointers | Legendary | Allows unit cloning and permanent buffs. |
| **Recursion** | Infinite Loops | Legendary | Re-executes the last 3 turns of action logs. |
| **Concurrency** | Threads & Parallelism | Legendary | Doubles unit count for 5 turns. |
| **Immutability** | Pure Functions | Epic | Units become immune to debuffs. |
| **Abstraction** | Interfaces & Types | Epic | Forces all enemy units to switch factions for 1 turn. |
| **Compilation** | Build Cycles | Epic | Instantly heals all friendly units to max. |
| **Entropy** | Randomness | Rare | Shuffles all unit positions globally. |
| **State** | Mutability | Rare | Allows stealing a buff from an enemy. |
| **Pattern** | Regex & Matching | Rare | Deals bonus damage to specific unit types. |
| **Binding** | Closures | Uncommon | Links two enemy units; damage to one hits the other. |
| **Flow** | Control Logic | Uncommon | Forces an enemy to retreat 3 spaces. |
| **Protocol** | Contracts & RPC | Uncommon | Prevents one enemy unit from using abilities. |
| **Origin** | The Source Code | Mythic | Grants total control of the Arena for 3 turns. |
